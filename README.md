# B-Rating 综合评分计算器

![Version](https://img.shields.io/badge/Version-27.3-ff2d55) ![License](https://img.shields.io/badge/License-MIT-green) ![Platform](https://img.shields.io/badge/Platform-Web-blue)

**B-Rating** 是一款专为音乐游戏《BanG Dream!少女乐团派对》玩家设计的综合实力评分工具。不同于传统的单一等级计算，它引入了**基础实力 (P)** 与 **综合深度 (G)** 的双维度评价体系，旨在更客观地反映玩家的真实水平与游戏完成度。

---

## 🌟 核心机制介绍

计算器遵循以下核心公式，总评分上限为 **31.0**：

$$Total\ Rating = \text{Min} \left( \frac{\sum \text{Best 30 Scores}}{30} + \frac{pt}{pt + 2500}, \ 31.0 \right)$$

### 1. 基础实力分 (P-Rating)
* **计算方式**：选取玩家表现最好的 30 场成绩（Best 30），根据歌曲等级与判定系数计算算术平均值。
* **判定系数**：
    * **ALL PERFECT (AP)**: 1.04
    * **FULL COMBO (FC)**: 1.01
    * **CLEAR**: 0.90
* **作用**：决定了玩家评分的“大区段”，是实力的硬性指标。

### 2. 综合深度分 (G-Rating)
* **点数化 (pt)**：统计所有难度的完成情况：
    * **EXPERT/SPECIAL**: 每个 FC+3pt，每个 AP 再+3pt
    * **HARD**: 每个 FC+1pt，每个 AP 再+1pt
    * **NORMAL**: 每个 FC+0.5pt，每个 AP 再+0.5pt
    * **EASY**: 每个 FC+0.2pt，每个 AP 再+0.2pt
* **非线性收益**：采用 $\frac{pt}{pt + 2500}$ 公式，随着点数增加，收益递减。
* **作用**：反映玩家对游戏内容的“完成度”与全难度制霸能力，作为实力的精修补强。

## 🛠️ 网页设计与功能特性

### 🎨 舒适的视觉体验
* **动态渐变背景**：采用流体渐变动画，视觉效果灵动且具有节奏感。
* **毛玻璃特效 (Glassmorphism)**：界面采用半透明模糊背景，兼顾美观与文字可读性。
* **个性化卡片**：支持自定义玩家颜色，并可**上传背景图片**，支持缩放与位移调整，生成独一无二的成绩单。

### 📱 移动端适配优化
* **响应式布局**：针对手机端优化了输入体验。B30 录入行在移动端自动切换为 **2x2 网格布局**，更符合触控操作。
* **防遮挡设计**：作者信息与功能按钮在移动端重新定位，确保在小屏幕上操作流畅。
* **锁定比例**：成绩预览卡片锁定 **480:850** 黄金比例，确保在任何设备上截图分享都不会变形。

### 💾 数据管理与持久化
* **全自动本地化**：开启历史记录后，所有录入的 B30 数据、G 组分数、玩家配置均会自动保存至浏览器 `LocalStorage`，**刷新不丢失**。
* **趋势图表**：内置 Chart.js 生成的评分变化曲线，直观回顾你的成长历程。
* **导入导出**：支持 JSON 格式数据备份，方便跨设备转移数据。

## 🚀 快速开始

1. **录入数据**：在“基础实力”栏输入你的 B30 成绩，在“综合深度”栏填入全难度计数。
2. **个性化定制**：上传背景，调整缩放和偏移。
3. **生成结果**：点击“🚀 计算并生成最终总成绩图”。
4. **保存分享**：点击“📥 保存成绩图到本地”，一键下载高清 PNG。
